# global-reach-landing

Portfólio pessoal de Davi Thiesse — SPA em Vite + React + Tailwind.

## Stack
- **Framework**: Vite + React + TypeScript
- **Styling**: Tailwind CSS + shadcn/ui
- **Deploy**: Vercel (www.thiesse.com.br)
- **Routing**: React Router (SPA, rewrites via vercel.json)

## Estrutura
```
src/
├── pages/
│   ├── Index.tsx          # Página principal / portfólio
│   ├── MissionControl.tsx # Landing page do Mission Control
│   └── NotFound.tsx
public/
├── og.png                 # OG image raiz (1200x630)
└── og-mission-control.png # OG image /mission-control
```
